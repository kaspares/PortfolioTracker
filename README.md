# StocksTracker

> **⚠️ WORK IN PROGRESS** 

A personal stock portfolio tracking web app built with Vue 3 and TypeScript. Designed to give a clean overview of your investments, signals, and portfolio performance.

## Features (planned / in progress)

- Dashboard with portfolio overview and charts
- Portfolio management view
- Trading signals panel
- Notifications system
- User authentication
- Customizable settings

## Tech Stack

- [Vue 3](https://vuejs.org/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) — build tool
- [Pinia](https://pinia.vuejs.org/) — state management
- [Vue Router](https://router.vuejs.org/)
- [Chart.js](https://www.chartjs.org/) + [vue-chartjs](https://vue-chartjs.org/)
- [Tailwind CSS v4](https://tailwindcss.com/) + [DaisyUI](https://daisyui.com/)

## Getting Started

```sh
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```

## Requirements

- Node.js `^20.19.0` or `>=22.12.0`
- pnpm
